MRHeart

Description: Cardiac MRI plugin - calculate end diastolic, end systolic and stroke volumes, ejection fraction and cardiac mass from cardiac MRI CINE data.

  • Choose volumetric CINE MRI data and run the plugin
  • Draw contours, label them as Left/Right Ventricle Epi/Endocardium using plugin buttons
  • Calculate the results
